# Anything - The Mobile "Everything" for Android

Tired of slow Android file search? Anything finds your PDFs, Word docs, and files instantly — just like Everything on Windows.

## Features
- Lightning-fast filename & content search
- No root required
- Lightweight & privacy-friendly

## Download
- [Anything_v1.0.apk](直接上传文件后生成的链接)

## How to install
1. Download the APK
2. Enable "Install from unknown sources" in Settings > Security
3. Install & enjoy!
